8086 的地址总线是 20 位宽，也就是其寻址范围是 2 的 20 次方=1MB。但其内部寄存器都是 16 位的

+--------------------------+  高地址
|        栈区 (Stack)       |
|  - 局部变量              |
|  - 函数调用帧            |
|  - 返回地址              |
+--------------------------+
|        堆区 (Heap)        |
|  - 动态分配 (malloc/new) |
|  - 用户控制增长/释放     |
+--------------------------+
|   未初始化数据段 (BSS)    |
|  - 未赋值的全局/静态变量 |
+--------------------------+
|   已初始化数据段 (Data)   |
|  - 已赋值的全局/静态变量 |
+--------------------------+
|    代码段 (Text/Code)     |
|  - 程序机器指令          |
|  - 只读常量（部分编译器）|
+--------------------------+  低地址

┌────────────────────┐ 高地址
│ 栈区 (Stack)       │
│ 堆区 (Heap)        │
│ 静态区 (Static)    │ → 数据段 + BSS 段
│ 常量区 (Const)     │ → 只读常量、字符串常量
│ 代码区 (Text)      │ → 程序指令
└────────────────────┘ 低地址
