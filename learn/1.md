1. 通用寄存器 ← 段寄存器 ✅ 合法

段寄存器的值可以读到通用寄存器里：

mov ax, cs   ; ✅ 合法
mov bx, ds   ; ✅ 合法
mov cx, ss   ; ✅ 合法
mov dx, es   ; ✅ 合法




2. 段寄存器 ← 通用寄存器 ✅ 合法

通用寄存器的值可以写进段寄存器：
mov ds, ax   ; ✅ 合法
mov es, bx   ; ✅ 合法
mov ss, cx   ; ✅ 合法
mov ds, dx   ; ✅ 合法





段寄存器 ← 段寄存器 ❌ 不合法
mov ds, cs   ; ❌ 非法
mov es, ds   ; ❌ 非法

mov ax, cs
mov ds, ax


4. 段寄存器 ← 立即数 ❌ 不合法
mov ds, 0x1234   ; ❌ 错误

mov ax, 0x1234
mov ds, ax


| 类别    | 寄存器            | 作用               |
| ----- | -------------- | ---------------- |
| 通用寄存器 | AX, BX, CX, DX | 存数据/操作数，可分高低 8 位 |
| 段寄存器  | CS, DS, SS, ES | 分段寻址             |
| 指针寄存器 | IP, SP, BP     | 程序计数/栈/基址        |
| 变址寄存器 | SI, DI         | 数组、字符串处理         |
| 标志寄存器 | FLAGS          | 状态与控制位           |


| 方式            | 适用阶段           | 优缺点              |
| ------------- | -------------- | ---------------- |
| BIOS 中断打印     | MBR / 引导 / DOS | 简单易用，慢，功能有限      |
| 绕过 BIOS 直接写显存 | 操作系统 / 内核      | 快，自由控制，复杂，需要硬件知识 |


| 寄存器     | 全称                     | 含义                      |
| ------- | ---------------------- | ----------------------- |
| **ESP** | Extended Stack Pointer | 栈指针寄存器（始终指向**当前栈顶**）    |
| **EBP** | Extended Base Pointer  | 栈基址寄存器（通常指向**当前函数的栈底**） |
